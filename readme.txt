== Description ==
This is a Woocomerce based plugin that will allow users to create custom professional checkout pages. This plugin allows users to create sidebar content within the Woocommerce checkout pages with text, images, and checklist/bullet points. We also made it easy to change the colors, fonts, and layout style of the checkout page to help match the user's site.

[youtube https://www.youtube.com/watch?v=fvd7MZeH9M0]

= KEY FEATURES =
- Custom layout style
- Accordion, Multistep and Tab layouts
- Checklist and Bullet Points
- Based highest converting checkout pages
- Custom text fields
- Custom sidebar images
- Custom colors
- Custom Google fonts
- Customer Header & Footer
- Custom Order confirmation page
- Mobile friendly
- Compatible with Stripe, PayPal, Authorize.NET, Square, and so on
- Fast loading
- Enhance security
- Easy setup


== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Custom Checkout Free, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type Custom Checkout Free and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

Before you activate this plugin, make sure you activate WooCommerce plugin.

= Manual installation =

The manual installation method involves downloading our eCommerce plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
